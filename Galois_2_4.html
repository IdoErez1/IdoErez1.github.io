2.4 הרחבה נורמלית

הרחבה נורמלית

הרחבה \(E / F\) תקרא נורמלית אם לכל איבר ב-\(E\) הפולינום המינימלי שלו מעל \(F\) מתפצל ב-\(E\).

ההרחבה \(\mathbb{Q}\left( \sqrt[3]{ 2 } \right) / \mathbb{Q}\) לא נורמלית. כאשר \(\alpha = \sqrt[3]{ 2 }\) נקבל \(m_{\alpha}=x^3 - 2\) ולכן \(\beta = \sqrt[3]{ 2 }e^{2\pi i/3} \not\in \mathbb{Q}\left( \sqrt[3]{ 2 } \right)\).

עבור הרחבה \(E / F\) סופית, מתקיים: $$\left\lvert \mathrm{Gal}( E / F) \right\rvert \leq \iota (E / F)$$ ושיוויון מתקיים אם"ם לכל שדה סגור אלגברית \(\Omega\) שמכיל את \(E\) ולכל שיכון \(\tau:E \to \Omega\) כך ש-\(I|_{F}=Id_{F}\) מתקיים \(\tau(E)=E\).

עבור הרחבה סופית התנאים הבאים שקולים

  1. ההרחבה היא נורמלית

  2. השדה \(E\) הוא שדה הפיצול של פולינום כלשהו \(P \in F[x]\).

  3. מתקיים \(\left\lvert \mathrm{Gal}(E / F) \right\rvert=\iota(E / F)\)

נניח ראשית כי ההרחבה היא נורמלית. כיוון שההרחבה היא סופית ניתן לכתוב \(E=F\left( \alpha_{1},\dots,\alpha_{k} \right)\) עבור \(\alpha_{i}\) אלגברים מעל \(F\). מההנחה נקבל כי \(m_{\alpha_{i}}\) מתפצל ב-\(E\) לכל \(i\). בפרט נקבל: $$P=\prod_{i=1}^{k}m_{\alpha_{i}}$$ מתפצל ב-\(E\). כמו כן לכל שדה \(F\subseteq K \subseteq \Omega\) המכיל את שורשי \(P\) מכיל גם את \(\alpha_{1},\dots,\alpha_{k}\) ולכן \(E\subseteq K\). כלומר \(E\) הוא שדה הפיצול של \(P\).

הרחבה היא נורמלית אם"ם יש שורש של \(p(x)\) גורר שיש את כל השורשים בהרחבה.

$$1 \leq Gal(L / F) \leq \iota( L / F) \leq [L : F]$$

1) מעל שדה סגור אלגברי, כל הרחבה היא נורמלית.

2) ההרחבה \(\mathbb{Q}\left( \sqrt{ 2 } \right) / \mathbb{Q}\) נורמלית

3) ההרחבה \(\mathbb{Q}\left( \sqrt[3]{ 2 } \right) / \mathbb{Q}\) לא נורמלית
